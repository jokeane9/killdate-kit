# Completed features

Finished features, kept for reference. Each folder contains the build artifacts that were locked before the build started.

A feature moves here when:
- All Cursor tasks are done and reviewed
- The deploy checklist has been run
- The feature is live in production and verified
- Any kill dates are recorded in `project-management/_log.md`

Don't delete these. The locked docs show what was decided and why. They're the evidence trail for future builds that touch the same surfaces.
