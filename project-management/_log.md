# Session log

One entry per session, most recent at the bottom. Claude Code reads the last 80 lines at session start — this is the primary context-transfer mechanism between sessions.

If work is mid-flight at session end, add a **RESUME HERE** block with numbered next steps. The next session picks up from there.

---

## Entry format

```
## YYYY-MM-DD

**Branch:** [branch name]
**What happened:** [2–3 sentences — what changed, what was decided, what shipped]

**RESUME HERE (if mid-flight):**
1. [next step]
2. [next step]
3. [next step]
```
