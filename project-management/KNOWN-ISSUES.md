# Known issues

Open bugs only. Fixed bugs live in git history. When a bug is fixed, remove it from this file — don't leave resolved entries.

---

## Bug ID prefixes

- `B-` — application logic
- `INT-` — integration (external API, webhook, third-party service)
- `CFG-` — configuration or environment
- `INF-` — infrastructure (deploy, database, networking)
- `P-` — performance
- `SEC-` — security

---

## Entry format

**[ID] Short description**
- **Severity:** low / medium / high
- **Symptoms:** [what you observe]
- **Suspected cause:** [if known, otherwise "unknown"]
- **Workaround:** [if any, otherwise "none"]
- **Opened:** [YYYY-MM-DD]
