# [AppName] — Product Canonical

> **Fill this in before your first session.** This is the document everything else derives from.
>
> The canonical has two jobs:
> 1. Answer "what IS this product?" for any new session, new contributor, or new AI tool
> 2. Be the document you argue FROM, not about. When a feature decision is disputed, this is the tiebreaker.
>
> Update it first. Propagate downstream when told to. Never build something that contradicts it without updating it first.
